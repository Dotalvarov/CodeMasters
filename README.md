# CodeMasters

CodeMasters es un videojuego educativo web diseñado para reforzar conocimientos básicos de programación mediante retos interactivos y dinámicas gamificadas.

El proyecto fue desarrollado utilizando HTML, CSS y JavaScript, incorporando almacenamiento local con IndexedDB y diseño responsive.

---

## Descripción

CodeMasters está estructurado en múltiples niveles interactivos que evalúan y fortalecen conceptos relacionados con Scratch y Micro:bit.

El juego incluye:

- Sistema de registro y login
- Pantalla de carga
- Persistencia de datos mediante IndexedDB
- Diseño adaptable (Responsive Design)
- Diferentes dinámicas de aprendizaje gamificado

---

## Niveles del juego

### Nivel 1 – Preguntas de selección múltiple
Preguntas relacionadas con Scratch.  
Subniveles: 1, 1.1 y 1.2.

### Nivel 2 – Arrastra y pega
Actividad interactiva para organizar correctamente diagramas de flujo.  
Subnivel: 2.1.

### Nivel 3 – Sopa de letras
Búsqueda de conceptos relacionados con:
- Scratch
- Micro:bit  
El jugador puede elegir la temática.

### Nivel 4 – Concéntrese
Juego de memoria basado en conceptos de Scratch.

### Nivel 5 – Rompecabezas
El jugador puede elegir entre:
- Rompecabezas del gato de Scratch
- Rompecabezas de la placa Micro:bit

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- IndexedDB (Base de datos del navegador)

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

   git clone https://github.com/Dotalvarov/CodeMasters.git

2. Ingresar al directorio del proyecto:

   cd CodeMasters

3. Abrir el archivo `pantalla de carga.html` en el navegador.
